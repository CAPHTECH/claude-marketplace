# Planning Plugin (Ideation, Product Planning & Task Planning)

アプリアイデアの具体化・プロダクト計画・タスク別プラン生成を支援するプラグインです。

## スキル一覧（3スキル）

| スキル | 説明 |
|--------|------|
| `app-idea-workshop` | アプリアイデアを共創型インタビューで具体化 |
| `rpt-vision-workshop` | RPTフレームワークでビジョン・戦略・優先順位を共創 |
| `task-plan-builder` | タスク種別に応じて適切な計画リファレンスを選び、実行可能なプランを組み立てる |

## インストール

```json
{
  "plugins": [
    "/path/to/claude-marketplace/planning-plugin"
  ]
}
```
