| int32 | -2,147,483,648 | 2,147,483,647 | 0, -1, 1 |
| int64 | -9,223,372,036,854,775,808 | 9,223,372,036,854,775,807 | 0, -1, 1 |
| uint32 | 0 | 4,294,967,295 | 0, 1 |

| decimal | 言語依存 | 言語依存 | 0, 0.01, 0.001（精度境界） |

## プロパティベーステストの性質カタログ

| 性質 | 説明 | 例 |
|------|------|---|
| 単調性 | f(a) ≤ f(b) if a ≤ b | ソート後の長さは変わらない |
| 可逆性 | decode(encode(x)) == x | シリアライズ/デシリアライズ |
| 冪等性 | f(f(x)) == f(x) | 正規化処理 |
| 保存則 | sum(before) == sum(after) | 在庫移動 |
| 交換律 | f(a, b) == f(b, a) | 加算、マージ |
| 結合律 | f(f(a, b), c) == f(a, f(b, c)) | 文字列結合 |
| 同一律 | f(x, identity) == x | 0との加算 |
| 分配律 | f(a, g(b, c)) == g(f(a, b), f(a, c)) | 計算処理 |
