# ユーザーストーリー

## 優先度の定義

- **Must**: MVPに必須
- **Should**: できれば含めたい
- **Could**: 余裕があれば
- **Won't**: 今回は対象外

## 見積の定義

- **S**: 1-2日
- **M**: 3-5日
- **L**: 1-2週間
- **XL**: 要分割
