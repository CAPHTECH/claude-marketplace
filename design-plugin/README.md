# Design Plugin (UI/UX Design)

Webアプリ・モバイルアプリのUI/UXデザインを体系的に行うプラグインです。

## スキル一覧（3スキル）

| スキル | 説明 |
|--------|------|
| `design-system-builder` | デザインシステムの構築・運用（Design Tokens 3層・DDR/QOC・ガバナンス・UIインベントリ） |
| `web-app-designer` | Webアプリの個別画面・コンポーネントのデザイン（Design Thinking・Atomic Design・グリッドシステム・WCAG 2.1） |
| `mobile-app-designer` | iOS/Androidアプリのデザイン（Apple HIG・Material Design 3準拠） |

## インストール

```json
{
  "plugins": [
    "/path/to/claude-marketplace/design-plugin"
  ]
}
```
