# Design Plugin (UI/UX Design)

Webアプリ・モバイルアプリのUI/UXデザインを体系的に行うプラグインです。

## スキル一覧（2スキル）

| スキル | 説明 |
|--------|------|
| `web-app-designer` | Webアプリのデザイン（Design Thinking・Atomic Design・グリッドシステム・WCAG 2.1） |
| `mobile-app-designer` | iOS/Androidアプリのデザイン（Apple HIG・Material Design 3準拠） |

## インストール

```json
{
  "plugins": [
    "/path/to/claude-marketplace/design-plugin"
  ]
}
```
